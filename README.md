# Alpha Club - Boutique NFT Telegram

🎮 Plateforme de vente de NFT intégrée directement dans Telegram

## 🚀 Fonctionnalités

- ✨ **Boutique NFT** - Catalogue de produits NFT
- 💰 **Paiements TON** - Paiements sécurisés en cryptomonnaie
- 🔐 **Authentification Telegram** - Connexion simplifiée
- 📱 **Responsive** - Optimisé pour mobile
- ⚡ **Déployé sur Vercel** - Accès rapide

## 📋 Prérequis

- Compte Telegram Bot (@BotFather)
- Wallet TON
- Compte Vercel

## ⚙️ Installation

1. **Clone le repo**
   ```bash
   git clone https://github.com/ledomainedesakura/Testprojettg.git
   ```

2. **Configure les variables d'environnement**
   ```
   TELEGRAM_BOT_TOKEN=xxx
   TON_API_KEY=xxx
   ```

3. **Déploie sur Vercel**
   ```bash
   vercel
   ```

## 🎯 Utilisation

1. Accédez à votre Mini App Telegram
2. Parcourez les NFT disponibles
3. Connectez votre wallet
4. Achetez les NFT avec TON

## 📝 Configuration Bot Telegram

Dans @BotFather:
```
/newapp
Nom: Alpha Club
App Short Name: alphaclub
App Description: Boutique NFT Telegram
App URL: https://votre-app.vercel.app
```
